Aplikasi MentalKita 

MentalKita adalah sebuah aplikasi yang dibuat dengan tujuan untuk mengecek dan mengobati kesehatan mental. MentalKita memiliki 3 fitur utama yaitu fitur konseling, moodtracker, dan chatbot. Dengan adanya aplikasi MentalKita diharapkan dapat membantu masyarakat dalam mensejahterakan mentalnya.

## Instalasi

1. Pastikan Anda memiliki Python dan pip diinstal di sistem Anda.
2. Buka terminal atau command prompt.
3. Pindahkan ke direktori utama proyek.
4. Jalankan perintah berikut untuk menginstal dependensi:

    pip install .

    Jika modul-modul yang dibutuhkan tidak secara otomatis terinstal, anda dapat menambahkan flag -r atau --no-deps ke perintah di atas:

    pip install . --no-deps

## Penggunaan

1. Pastikan anda telah menginstall modul-modul yang dibutuhkan
2. Buka terminal atau command prompt.
3. Pindahkan ke direktori utama proyek.
4. Jalankan perintah berikut untuk menjalankan aplikasi:

    python main.py

5. Anda akan masuk ke menu awal. Disana anda dapat memilih untuk registrasi, login, atau keluar dari program
6. jika registrasi, maka anda dapat mendaftarkan username dan password yang nantinya akan disimpan ke database
7. jika login, maka anda dapat memasukkan username dan password dan nanti jika sesuai dengan data, maka anda dapat masuk ke halaman utama aplikasi MentalKita
8. ketika anda berhasil masuk ke halaman utama anda dapat memilih fitur yang tersedia, yaitu konseling, moodtracker, chatbot, dan riwayat konseling

# Cara penggunaan fitur konseling
1. Setelah berhasil masuk ke halaman utama, maka anda pilih masuk ke fitur konseling
2. Pilih metode konseling yang anda inginkan
3. Cari dan pilih psikolog yang tersedia
4. Lihat profil dan pesan sesi psikolog
5. Lakukan pembayaran dan konfirmasi pemesanan
6. Data pemesanan anda akan disimpan ke database

# Cara penggunaan fitur moodtracker
1. Setelah berhasil masuk ke halaman utama, maka anda pilih masuk ke fitur moodtracker
2. Anda dapat memilih mengisi data, melihat statistik mood dalam beberapa hari terakhir, atau melihat timeline moodtracker
2. Pertama pilih fitur isi data moodtracker
3. Pilih perasaan dan emosi yang menggambarkanmu pada hari ini
4. Cerita tentang apa yang terjadi hari ini
5. Simpan data moodtracker
6. Anda selanjutnya dapat melihat statistik mood yang telah diproses dari data yang telah Anda masukkan
7. Anda juga dapat melihat timeline dari moodtracker yang telah Anda isi sebelumnya

Cara penggunaan fitur chatbot
1. Setelah berhasil masuk ke halaman utama, maka anda pilih masuk ke fitur chatbot
2. Pilih kata kunci mengenai informasi kesehatan mental yang anda ingin cari
3. Lihat penjelasan dan artikel yang tersedia
4. Anda juga dapat mencari informasi lainnya yang tersedia
